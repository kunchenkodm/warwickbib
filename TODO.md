# Future Features & Backlog

- **Legacy BibTeX Support**: The upstream `alex-ball/bathbib` repository included a `bath-bst.dtx` file inside the `bst/` directory to generate a legacy `.bst` style for older BibTeX workflows. This fork currently focuses entirely on `biblatex` and CSL. As a future enhancement, we should consider writing a `warwick-bst.dtx` equivalent to populate the `bst/` directory, providing fallback support for users who cannot migrate to biblatex.
