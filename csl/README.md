# Cite Them Right / Warwick Harvard CSL

This directory contains the Citation Style Language (CSL) implementation of the Warwick/CTR Harvard referencing style.

The style file `harvard-warwick.csl` (or `harvard-ctr.csl` in the Standard branches) is used to format citations automatically in reference managers like **Zotero**, **Mendeley**, and **Microsoft Word**.

## Installation

You can download the pre-compiled CSL file from the [GitHub Releases](../../releases) page in the `biblatex-csl-package-*.zip` bundles.

1. Download the ZIP file for your required version (12th edition or 13th edition, Warwick or Standard).
2. Extract the ZIP file.
3. Open Zotero, go to **Settings > Cite > Styles**.
4. Click the `+` button and select the `.csl` file to install it.

## Maintaining the Style

The CSL file uses conditional XML formatting to implement specific overrides.
For detailed instructions on how the entire package works, please see the [main README.md](../README.md) at the root of the repository.
